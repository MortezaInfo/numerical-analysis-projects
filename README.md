# پروژه ۳ - محاسبات عددی و تحلیل داده

این پروژه شامل سه بخش اصلی است:

1. **درونیابی عددی**
   - روش لاگرانژ
   - اسپلاین مکعبی طبیعی

2. **تحلیل داده‌های فایل Excel**
   - خواندن داده‌ها به‌صورت پویا
   - رسم نمودارهای میله‌ای، خطی و جعبه‌ای
   - محاسبه میانگین زمان اجرا

3. **انتگرال‌گیری عددی**
   - روش ذوزنقه‌ای
   - روش سیمسون
   - روش انتگرال‌گیری گوسی
